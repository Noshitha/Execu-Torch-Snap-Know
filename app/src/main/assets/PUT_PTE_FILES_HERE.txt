Place face embedding model files here before building the APK:

  face_embedding.pte   ← generated by: python model_tools/export_face_embedding.py --qnn

Speech assets now live under structured subdirectories:

  speech/stt/whisper-tiny/                  ← Whisper tiny encoder/decoder bundle
  speech/tts/piper/en_US-lessac-medium/     ← Piper voice bundle

Use:
  python model_tools/export_whisper_tiny.py
  bash scripts/stage_speech_assets.sh --help

See README.md and MODEL_BUILD_GUIDE.md for full instructions.
